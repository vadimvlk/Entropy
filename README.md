# ENTROPY — Random Walk Terminal

Веб-приложение «случайная величина»: непрерывный поток случайных тиков (как сделки),
из которых в реальном времени строятся японские свечи OHLC на нескольких таймфреймах —
в стиле TradingView. График полностью случайный (броуновское блуждание), может уходить
в минус, сохраняется между запусками и продолжается с того же места.

Построено на **TradingView Lightweight Charts** (официальная бесплатная библиотека TradingView)
с собственным генератором тиков.

## Запуск

```bash
npm install
npm run dev      # http://127.0.0.1:5173
```

Продакшн-сборка:

```bash
npm run build
npm run preview  # http://127.0.0.1:4173
```

> Порт фиксированный — это держит origin стабильным, поэтому история в `localStorage`
> переживает перезапуск и график продолжается.

## Возможности

- **Поток тиков со случайной скоростью** — режимы `burst / fast / steady / slow / sparse`.
  В режиме **АВТО** меняются сами (то быстрее, то медленнее, с разрывами при «низкой ликвидности»);
  в режиме **РУЧ** скорость и волатильность задаёте вы через выпадающие списки.
- **Случайное блуждание** — аддитивное (без дрейфа), масштаб шага зависит от волатильности
  (`quiet / normal / active / wild`). Цена может уйти в минус.
- **Японские свечи OHLC** + типы графика: свечи / линия / область.
- **Объём** — гистограмма внизу (отдельная панель): число тиков на свечу, бары окрашены вверх/вниз.
- **Таймфреймы**: 1с, 5с, 15с, 1м, 5м. База — 1-секундные свечи, старшие ТФ агрегируются на лету.
- **Светлая и тёмная темы** — переключатель в шапке (солнце/луна), тема сохраняется.
- **Инструменты**: перекрестие (+ «магнит»), горизонтальная линия, вертикальная линия,
  **Фибоначчи 0/50/100/200%** (ставится протягиванием мыши и двигается drag&drop прямо на графике),
  стирание, очистка всех. Подсказки на всех кнопках при наведении.
- **Масштаб**: колесо / перетаскивание, приблизить/отдалить, «вместить всё», сброс.
- **Сохранение и продолжение** — свечи, объём, состояние генератора, режим, тема, нарисованные
  линии и Фибоначчи пишутся в `localStorage`. После закрытия/открытия график продолжается без разрыва.
- **Пауза/старт** генератора, **сброс** (новая случайная последовательность).

## Архитектура

| Файл | Назначение |
|------|-----------|
| `src/engine.ts` | Генератор тиков, режимы авто/ручной, блуждание, объём, агрегация ТФ |
| `src/chart.ts` | Контроллер Lightweight Charts: серии, объём, темы, инструменты, зум |
| `src/verticalLine.ts` | Кастомный примитив вертикальной линии |
| `src/fib.ts` | Кастомный примитив Фибоначчи (0/50/100/200%) |
| `src/storage.ts` | Персистентность (свечи+объём, состояние, линии/фибо, тема/режим/вид) |
| `src/rng.ts` | Гауссов сэмплер и помощники |
| `src/main.ts` | Связывание движка, графика и UI |
| `src/style.css` | Дизайн «промышленный трейдинг-терминал» + светлая тема |

## Удалённое управление

Приложение полностью работает в браузере. Состояние симуляции и рисунков лежит в `localStorage`
под ключами `random-walk-terminal:*` — их можно читать/менять для интеграции.
