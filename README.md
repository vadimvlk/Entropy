# ENTROPY — Random Walk Terminal

Веб-приложение «случайная величина»: непрерывный поток случайных тиков (как сделки),
из которых в реальном времени строятся японские свечи OHLC на нескольких таймфреймах — в
стиле TradingView, с торговлей, объёмом и инструментами. Цена движется как криптоактив
(всегда положительна, может стремиться к нулю), график сохраняется между запусками и
продолжается с того же места.

Построено на **TradingView Lightweight Charts** (официальная бесплатная библиотека TradingView)
с собственным генератором тиков. Стек: **Vite + TypeScript**, без бэкенда — всё в браузере.

## Запуск

```bash
npm install
npm run dev        # http://127.0.0.1:5173
```

Продакшн-сборка и тесты:

```bash
npm run build      # tsc + vite build
npm run preview    # http://127.0.0.1:4173
npm run test:unit  # юнит-тесты логики (агрегация, объём, фибо, торговый движок)
npm run test:e2e   # headless E2E в браузере (нужен запущенный dev-сервер)
```

> Порт фиксированный — это держит origin стабильным, поэтому состояние в `localStorage`
> переживает перезапуск и график/счёт продолжаются.

## Возможности

### График и генератор
- **Поток тиков со случайной скоростью** — режимы `burst / fast / steady / slow / sparse`.
  В режиме **АВТО** меняются сами (то быстрее, то медленнее, с разрывами при «низкой ликвидности»);
  в режиме **РУЧ** скорость и волатильность задаёте вы через выпадающие списки.
- **Модель цены** — мультипликативное (геометрическое) случайное блуждание, как крипта:
  цена всегда положительна, без дрейфа, может приближаться к нулю (минимум `1e-7`), но **не уходит
  в минус**. Масштаб шага задаёт волатильность (`quiet / normal / active / wild`).
  Формат цены адаптивный: `1,012.34` … `0.00000010`.
- **Японские свечи OHLC** + типы графика: свечи / линия / область.
- **Объём** — гистограмма внизу (отдельная панель): число тиков на свечу, бары окрашены вверх/вниз.
- **Таймфреймы**: 1с, 5с, 15с, 1м, 5м. База — 1-секундные свечи, старшие ТФ агрегируются на лету.
- **High / Low видимого участка** — нейтральные метки на правой оси цены (как метка текущей цены),
  пересчитываются при зуме и сдвиге.
- **Светлая и тёмная темы** — переключатель в шапке (солнце/луна), тема сохраняется.
- **Масштаб**: колесо / перетаскивание, приблизить/отдалить, «вместить всё», сброс.

### Торговля
- Рыночные ордера **Buy / Sell**, плечо **1:10**, контракт шаг/мин **0.1**, депозит при старте **$1000**.
- Нетто-позиция с разворотом: лонг 1 контракт − продажа 2 = **шорт 1**.
- Средняя цена входа, **реализованный** и **нереализованный** P&L (в $ и %), маржа и свободные средства.
- **Ликвидация** при equity ≤ 50% маржи — баланс не уходит в минус. Перебор плеча отклоняется.
- На графике: линии **входа** и **ликвидации**, маркеры сделок (стрелки Buy/Sell).
- Кнопка «Закрыть позицию», степпер количества с пресетами 25/50/75/MAX. Счёт сохраняется.

### Инструменты
- Перекрестие (+ режим «магнит» с привязкой к свечам).
- Горизонтальная и вертикальная линии.
- **Фибоначчи 0 / 50 / 100 / 200%** — ставится протягиванием мыши, тянется **только вправо** от точки
  постановки (не в историю); двигается захватом самих уровней (0% / 100% — края, 50% / 200% — вся фиба).
- Стирание линий, очистка всех. Подсказки на всех кнопках при наведении.

### Сохранение
Свечи + объём, состояние генератора, режим, тема, выбранный вид, нарисованные линии, Фибоначчи и
торговый счёт пишутся в `localStorage`. После закрытия/открытия всё продолжается без разрыва.
Загружаемое состояние валидируется — повреждённая запись отбрасывается и стартует новый график.

## Архитектура

| Файл | Назначение |
|------|-----------|
| `src/engine.ts` | Генератор тиков (GBM), режимы авто/ручной, объём, агрегация таймфреймов |
| `src/trading.ts` | Торговый счёт: неттинг с разворотом, P&L, маржа, плечо, ликвидация |
| `src/chart.ts` | Контроллер Lightweight Charts: серии, объём, темы, инструменты, позиция, High/Low, зум |
| `src/fib.ts` | Кастомный примитив Фибоначчи (0/50/100/200%, якорь по времени, вправо) |
| `src/verticalLine.ts` | Кастомный примитив вертикальной линии |
| `src/hilo.ts` | Примитив меток High/Low видимого участка на оси цены |
| `src/format.ts` | Адаптивное форматирование цены/денег |
| `src/storage.ts` | Персистентность (свечи+объём, состояние, счёт, линии/фибо, тема/режим/вид) |
| `src/rng.ts` | Гауссов сэмплер и помощники |
| `src/main.ts` | Связывание движка, графика, торговли и UI |
| `src/style.css` | Дизайн «промышленный трейдинг-терминал» + светлая тема |

## Удалённое управление

Приложение полностью работает в браузере. Состояние симуляции, счёта и рисунков лежит в
`localStorage` под ключами `random-walk-terminal:*` — их можно читать/менять для интеграции.
